**If you need to install the shapes3D library, select Sketch -> Import Library... -> Add Library...
and then seach from "Shapes 3D" by Peter Lager.**

When you run skeletonV2 you will be prompted with a file select menu.  
Select a BVH file from the motion data folder and the program will load a skelton to display it.

If you want to load a new motion data file, press 3 from within the application and you will be 
able to select a new BVH file from the file select menu.

You can also control the camera/perspetive using the w/s keys to zoom in/out and the a/d keys 
to pan to the side.
